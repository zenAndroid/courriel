Interface web pour un système de messagerie (courriel)

Courriel est un système de messagerie pair-à-pair, c.-à-d., sans serveur centrale.

Le projet courriel est séparé en deux parties:

  1. Projet 1: Interface graphique pour le courriel
    2. (code en html/css/js à exécuter par un navigateur).
  2. Projet 2: Un mini-serveur permettant le partage (distribution)
    3. des courriels (messages) entre plusieurs utilisateurs.

L'interface WEB devrait pouvoir:

  1. Visualiser la liste de messages
  2. Composer un nouveau message.
  3. Visualiser et rechercher le carnet d'adresse
